# 10 రోజుల కుటుంబ ఆహార మెనూ 🍽️ (పూర్తి పట్టిక)

కుటుంబం: నాన్న (37), అమ్మ (29), అమ్మాయి (9), అబ్బాయి (5) · **ఆంధ్ర శైలి**
నియమం: ఉదయం & రాత్రి **టిఫిన్** (అన్నం వద్దు), మధ్యాహ్నం మాత్రమే **అన్నం**.
నాన్‌వెజ్: **మంగళ, బుధ, ఆది** — ఉదయం + మధ్యాహ్నం మాత్రమే; ఆ రోజుల రాత్రి తేలికైన వెజ్.

| **7** ఆది | **నాన్‌వెజ్** | చికెన్ కీమా + దోస | ఆంధ్ర మటన్ కూర + అన్నం + ఉల్లి చారు | డ్రైఫ్రూట్స్ + కొబ్బరి నీళ్లు | పుల్కా + మజ్జిగ పులుసు *(తేలిక)* |

| **9** మంగళ | **నాన్‌వెజ్** | ఎగ్ కర్రీ + ఇడ్లీ | ఆంధ్ర కోడి వేపుడు + అన్నం + చారు | మొలకలు + పండు | గోంగూర పప్పు + చపాతీ *(తేలిక)* |
| **10** బుధ | **నాన్‌వెజ్** | ఎగ్ పులుసు + పుల్కా | రొయ్యల ఇగురు + అన్నం | చిలగడదుంప + పల్లీలు | మిల్లెట్ ఇడ్లీ + చట్నీ *(తేలిక)* |

## ✅ ప్రతిరోజూ వీటిని తప్పక చేర్చండి (డాక్టర్ సూచనలు)

*పైన మెనూతో పాటు ఈ క్రింది వాటిని ప్రతిరోజూ ఇవ్వడం వల్ల పోషకాహారం పూర్తవుతుంది.*

| అంశం | ఏమి / ఎంత | ఎవరికి | ఎందుకు |

| ☀️ ఉదయం పరగడుపున | 4–5 నానబెట్టిన బాదం + 1 ఖర్జూరం | పిల్లలు | మెదడు ఎదుగుదల, ఐరన్ |
| 🍎 పండు | రోజూ 1 (సీజనల్: అరటి, యాపిల్, దానిమ్మ, బొప్పాయి, జామ, బత్తాయి, మామిడి) | అందరూ | విటమిన్లు, ఫైబర్ |
| 🥛 పాలు | ఉదయం 1 + రాత్రి పడుకునే ముందు 1 గ్లాసు | పిల్లలు | కాల్షియం, ఎదుగుదల, నిద్ర |
| 🥗 మధ్యాహ్నం సలాడ్ | దోసకాయ, క్యారెట్, బీట్‌రూట్, ఉల్లి, నిమ్మ | అందరూ | ఫైబర్, విటమిన్లు, ఐరన్ గ్రహణ |
| 🥣 పెరుగు/మజ్జిగ | మధ్యాహ్నం భోజనంతో | అందరూ | జీర్ణం, గట్ ఆరోగ్యం |

| 🍎 రోజువారీ పండు (గతంలో 1 రోజే) | "ప్రతిరోజూ తప్పక" పట్టికలో సీజనల్ పండు చేర్చాం |
| 🥛 పిల్లల పాలు (అస్థిరంగా) | రోజూ 2 గ్లాసులు (ఉదయం + రాత్రి) స్థిరం చేశాం |

| 🌞 వేసవి హైడ్రేషన్ (జూన్) | కొబ్బరి నీళ్లు/మజ్జిగ చేర్చాం |

### ఇప్పటికే బాగున్నవి 👍
ప్రతి భోజనంలో ప్రోటీన్ (పప్పులు, మొలకలు, పనీర్, సోయా, గుడ్డు/మాంసం/చేప) · ఒమేగా-3 (చేప) · ఐరన్ + విటమిన్-C (గోంగూర, పాలకూర, నిమ్మ) · జీర్ణం (పెరుగు/మజ్జిగ) · తృణధాన్యాలు (రాగి, మిల్లెట్, ఓట్స్) · వైవిధ్యం (ఏ వంటకం రిపీట్ కాదు).

## 🎒 పిల్లల స్కూల్ బాక్స్ (సోమ–శుక్ర)

పిల్లలు **ఇంట్లో ఉదయం టిఫిన్ తిని** స్కూల్‌కి వెళ్తారు; కింది బాక్స్‌లు తీసుకెళ్తారు.
- **అబ్బాయి (5 ఏళ్లు):** స్కూల్‌కి ఉదయం స్నాక్ + లంచ్; **సాయంత్రం స్నాక్ ఇంట్లో** (స్కూల్ నుంచి వచ్చాక).
- **అమ్మాయి (9 ఏళ్లు):** స్కూల్‌కి ఉదయం స్నాక్ + లంచ్ + సాయంత్రం స్నాక్.
- *అబ్బాయికి అదే పదార్థాలు, కొంచెం తక్కువ పరిమాణం.*
- స్కూల్ రోజులు: **రోజు 1–5 (సోమ–శుక్ర), రోజు 8–10 (సోమ–బుధ)**. శని, ఆది (రోజు 6–7) స్కూల్ లేదు → కుటుంబ మెనూ ఇంట్లోనే.

*ఉదయం & సాయంత్రం **స్నాక్ = 10 నిమిషాల బ్రేక్**: చిన్న, చేతితో తినే **ఇంట్లో చేసిన పిండి వంటకం + పండు** మాత్రమే (టిఫిన్/అన్నం కాదు). లంచ్ మాత్రం పూర్తి భోజనం.*

| రోజు | 🎒 ఉదయం స్నాక్ (10 ని.) — పిండి వంటకం + పండు | 🍱 లంచ్ బాక్స్ | 🌆 సాయంత్రం స్నాక్ (ఇద్దరికీ) — అమ్మాయి స్కూల్‌లో, అబ్బాయి ఇంట్లో |
|:---:|:---|:---|:---|
| **1** సోమ | రాగి బిస్కెట్లు + అరటిపండు | పాలకూర పప్పు + అన్నం + పెరుగు *(కుటుంబ లంచ్‌లోదే)* | చెక్కలు (మైల్డ్) + ఖర్జూరం |
| **2** మంగళ | సున్ని ఉండలు + యాపిల్ ముక్కలు | పెరుగు అన్నం + క్యారెట్ ముక్కలు *(త్వరగా)* | పల్లీ-బెల్లం ఉండ + కొన్ని బాదం |
| **3** బుధ | పల్లీ చిక్కీ + దానిమ్మ గింజలు | నిమ్మకాయ అన్నం + పాపడ్ *(త్వరగా)* | జంతికలు (మైల్డ్) + యాపిల్ |
| **4** గురు | గోధుమ-బెల్లం బిస్కెట్లు + అరటిపండు | సెనగపప్పు సాంబారు + అన్నం + పెరుగు *(కుటుంబ లంచ్‌లోదే)* | అటుకుల మిక్చర్ (మైల్డ్) + అరటిపండు |
| **5** శుక్ర | రవ్వ లడ్డు + జామపండు | రాజ్మా + అన్నం + పెరుగు (మైల్డ్) *(కుటుంబ లంచ్‌లోదే)* | నిప్పట్లు + దానిమ్మ |
| **8** సోమ | నువ్వుల ఉండలు + యాపిల్ ముక్కలు | టమాటా పప్పు + అన్నం + పెరుగు *(కుటుంబ లంచ్‌లోదే)* | రాగి కుకీలు + ఖర్జూరం |
| **9** మంగళ | మరమరాల (బెల్లం) ఉండలు + అరటిపండు | కొబ్బరి అన్నం + వేయించిన శనగలు *(త్వరగా)* | అరిసెలు (చిన్నవి) + కొన్ని బాదం |
| **10** బుధ | మినప సున్నుండలు + బత్తాయి తొనలు | టమాటా అన్నం + ఉడికించిన బఠాణీ *(త్వరగా)* | బనానా చిప్స్ (ఇంట్లో) + యాపిల్ |

### 🍱 స్కూల్ బాక్స్ చిట్కాలు (డాక్టర్)
- **స్నాక్ = పిండి వంటకం + పండు** మాత్రమే (10 ని. బ్రేక్‌కి సరిపోయేలా చిన్నగా). ఉదాహరణ పిండి వంటకాలు: రాగి/గోధుమ బిస్కెట్లు, సున్ని ఉండలు, పల్లీ/నువ్వుల ఉండలు, రవ్వ లడ్డు, చెక్కలు, నిప్పట్లు, జంతికలు.
- వీటిలో **ప్రోటీన్/కాల్షియం** ఎక్కువ: సున్ని ఉండలు (మినప్పప్పు), పల్లీ చిక్కీ, నువ్వుల ఉండలు — చక్కెర బదులు **బెల్లం** వాడండి, నూనె తక్కువ.
- **కారం వద్దు.** లంచ్ బాక్స్‌కి: వేడి (జూన్) వల్ల **స్టీల్ బాక్స్**, పెరుగు అన్నంలో మజ్జిగ ఎక్కువ, పండు విడిగా.
- **నాన్‌వెజ్ ఏదీ స్కూల్‌కి వద్దు** (గుడ్డుతో సహా) — లంచ్‌లో **వెజ్ ప్రోటీన్** మాత్రమే: పప్పు, పెరుగు, శనగలు, బఠాణీ.
- **అమ్మ పనిని తగ్గించడానికి:** వెజ్ రోజుల్లో (1, 4, 5, 8) పిల్లల లంచ్ = **కుటుంబ లంచ్‌లోదే** — కారం కలపక ముందే పిల్లల వాటా తీసి బాక్స్‌లో పెట్టండి (విడిగా వండక్కర్లేదు). నాన్‌వెజ్ రోజుల్లో (2, 3, 9, 10) మాత్రం **త్వరగా అయ్యే పెరుగు/నిమ్మ/కొబ్బరి/టమాటా అన్నం** — విడిగా కూర వద్దు.
- ప్రతి స్నాక్‌లో **పండు**, లంచ్‌లో **కార్బ్ + ప్రోటీన్** ఉండేలా చూశాం. నీళ్ల సీసా తప్పనిసరి; ప్యాకెట్ చిప్స్/చాక్లెట్ తగ్గించండి.

## 👩‍🍳 ముందుగా చేసి దాచుకునే ఇంటి స్నాక్స్ (బ్యాచ్ ప్రిప్)

*వారాంతంలో (శని/ఆది) ఒకేసారి చేసి **గాలి చొరని డబ్బాల్లో** దాచుకుంటే రోజూ హడావిడి ఉండదు.*

| స్నాక్ | ప్రధాన పదార్థాలు | నిల్వ |
|:---|:---|:---:|
| సున్ని ఉండలు | మినప్పప్పు/పుట్నాలు + బెల్లం + నెయ్యి | 15 రోజులు |
| పల్లీ చిక్కీ | పల్లీలు + బెల్లం | 15 రోజులు |
| నువ్వుల ఉండలు | నువ్వులు + బెల్లం | 15 రోజులు |
| చెక్కలు / నిప్పట్లు | బియ్యం పిండి + శనగపిండి | 15 రోజులు |
| జంతికలు (మురుకులు) | బియ్యం పిండి + మినప పిండి | 15 రోజులు |
| రాగి / గోధుమ బిస్కెట్లు | రాగి/గోధుమ పిండి + బెల్లం + నెయ్యి | 10 రోజులు |
| రవ్వ లడ్డు | బొంబాయి రవ్వ + నెయ్యి + బెల్లం | 1 వారం |
| మరమరాల ఉండలు | మరమరాలు + బెల్లం | 1 వారం |
| అటుకుల మిక్చర్ | అటుకులు + పల్లీలు | 1 వారం |
| బనానా చిప్స్ | పచ్చి అరటికాయ | 1 వారం |

**చిట్కాలు:** చక్కెర బదులు **బెల్లం**; **నూనె తక్కువ** (వీలైతే ఓవెన్/తక్కువ నూనె); ప్రోటీన్ ఎక్కువ ఉండే సున్ని ఉండలు, పల్లీ చిక్కీ, నువ్వుల ఉండలుకి ప్రాధాన్యం. **పండ్లు మాత్రం రోజూ తాజాగా** కట్ చేసి పెట్టండి (ముందే కట్ చేయొద్దు).
